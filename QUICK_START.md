## 步骤 1: 创建仓库
1. 访问 https://github.com/new
2. 填写:
   - Repository name: `ces-az-index`
   - Description: `CES Innovation Awards Database 2024-2026`
   - 选择 `Public`
3. 点击 **Create repository**

---

## 步骤 2: 上传文件
1. 点击 **"uploading an existing file"**
2. 上传本文件夹中的所有文件:
   - index.html
   - README.md
   - LICENSE
   - DEPLOY.md
   - CNAME
   - data/ (整个文件夹)
3. 点击 **Commit changes**

---

## 步骤 3: 启用 GitHub Pages
1. 点击 **Settings** → **Pages**
2. Source 选择:
   - Branch: `main`
   - Folder: `/(root)`
3. 点击 **Save**
4. 等待 2 分钟

---

## 🎉 完成！

## 绑定自定义域名（可选）

1. 购买域名 `ces-az-index.com`

3. 在 GitHub Pages 设置中填写自定义域名
