🌐 **Live Demo**: https://alexzhuang-glitch.github.io/ces-az-index

## 📊 Overview

CES-AZ-INDEX is an interactive data visualization platform that analyzes CES Innovation Awards data from 2024 to 2026. The platform provides insights into:

- **1,432** award-winning products
- **82** companies across **12** countries
- **36** product categories
- Hot tracks analysis (AI, Robotics, XR, etc.)
- Regional distribution trends

## ✨ Features

- 📈 **Interactive Charts** - Visualize data with Chart.js
- 🔍 **Advanced Filtering** - Search by year, category, region, company
- 📱 **Responsive Design** - Works on desktop and mobile
- ⚡ **Fast Loading** - Static site with CDN optimization
- 🌍 **Global Access** - Hosted on GitHub Pages
